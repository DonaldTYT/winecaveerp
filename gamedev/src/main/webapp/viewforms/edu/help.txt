==========
Course Record
==========
1. Create new course
1.1. Add
1.2. Input all required fields
1.3. Pick Days checkbox
1.4. Click Generate schedule button will generate session based on Start Date, "Number of Session", Days, Time,  Duration.
       This step is optional. You can add schedule later.
1.5. Click Save button to insert new record

2. Delete schedule (session)
2.1 Open existing course record
2.2 Click the delete button on the session item
2.3 Click save

3. Add Student
3.1. Add Student (Basic, insert one by one)
3.1.1. Open existing course record
3.1.2. Student list, click add row, and choose target student
3.1.3. Click Save
3.2. Add Student (Advanced, batch insert)
3.2.1. Visit Student Record page
3.2.2. Click Add to Course button, choose target students. You can use quick filter, Student Query to shortlist protential student.
3.2.3. Click Proceed
3.2.4. Choose Course or enter Course Code
3.2.5. Click Confirm. It will add selected students to specific course.

4. Add recurring course
4.1. Open existing course record
4.2. Click Add new recurring course
4.3. It will display a pre-filled input form,
       Course Code - based on course code + "_" + increment_id
       Start Date - based on last session date 
       Carry Student - carry student of current course
       Generate schedule - same as "Create new Course" -> Generate schedule


==========
Student Record
==========
1. Create new course
1.1. Add
1.2. Input all required fields
1.3. Click Save button to insert new record

2. Add Student to Assessment Schedule
2.1 Add new Assessment Schedule under Assessment Schedule
2.2.1 You can use Student Query for shortlist student in waiting state. i.e. Status = Waiting
2.2.2 Click Search
2.3 Add to Asmt, select target student, click Proceed
2.4 Pick target assessment schedule, and click Confirm. 

3. Add Student to Course
3.1 Add new Course under Course Record
3.2.1 You can use Student Query for shortlist student. **Remark: Assessment Query is not available yet.
3.2.2 Click Search
3.3 Add to Course. select target student, click Proceed
3.4 Pick target Course and click Confirm. 

4. Payment  **REMARK: This feature is dev in progress. 
4.1 Open a Student Record Detail
4.2 Enter required fields
4.3 Pick the subscribed course for payment
4.4 Click Save
4.5 Click Print and Close 
4.6 You can reprint the receipt by using the Record Detail -> Paymet -> Preview
